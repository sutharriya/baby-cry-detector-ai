# Baby Cry Detector AI

AI-powered Baby Cry Detection system that analyzes baby cry audio and predicts the reason using Deep Learning.

This project uses a CNN + BiLSTM model with Mel Spectrogram feature extraction and is deployed using Flask. It can be integrated into Android apps, web apps, and baby monitoring systems.

## Features

- Detect baby cry reason automatically
- Supports multiple classes:
  - Hungry
  - Belly Pain
  - Discomfort
  - Tired
- Deep Learning model (CNN + BiLSTM)
- Flask web interface
- Android app integration supported
- Real-time prediction
- Easy API integration

## Tech Stack

- Python

- TensorFlow / Keras
- Librosa (audio processing)
- Mel Spectrogram
- HTML, CSS
